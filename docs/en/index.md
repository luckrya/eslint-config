### Packages included in this project

Please follow the links below for the packages you care about.

- [@luckrya/eslint-config-cli](./tools/cli) - A tool to quickly integrate code specifications for your project. It mainly uses the official configuration of @luckrya/eslint-config-\*, allowing developers to focus more on business development. Simple and efficient.
- [@luckrya/eslint-config-core](./tools/core) - Integration of some core rule sets, dedicated to general pure JavaScript projects, suitable for browser and Node side.
- [@luckrya/eslint-config-base](./tools/base) - Integration of some basic rulesets to support TypeScript on top of `@luckrya/eslint-config-core`.
- [@luckrya/eslint-config-vue](./tools/vue) - Integrate rulesets for Vue framework applications, support Vue.js on the basis of `@luckrya/eslint-config-base`.
- [@luckrya/eslint-config-react](./tools/react) - Ruleset integration for React framework applications, React.js support based on `@luckrya/eslint-config-base`.
