### 项目包含的工具包

请点击以下链接获取您关心的软件包。

- [@luckrya/eslint-config-cli](./tools/cli) - 为项目快速集成代码规范的工具。 主要使用 `@luckrya/eslint-config-*` 的官方配置；让开发者更加专注于业务开发、简单高效。
- [@luckrya/eslint-config-core](./tools/core) - 一些核心规则集的整合，专用于一般纯 JavaScript 项目，适用于浏览器端和 Node 端
- [@luckrya/eslint-config-base](./tools/base) - 一些基础规则集的整合，在 `@luckrya/eslint-config-core` 基础上支持 TypeScript
- [@luckrya/eslint-config-vue](./tools/vue) - 适配以 Vue 框架的应用的规则集整合，在 `@luckrya/eslint-config-base` 基础上支持 Vue.js
- [@luckrya/eslint-config-react](./tools/react) - 适配以 React 框架的应用的规则集整合，在 `@luckrya/eslint-config-base` 基础上支持 React.js
