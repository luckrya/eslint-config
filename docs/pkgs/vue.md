## 安装

```sh
$ pnpm add -D @rc5/eslint-config-vue
```

## 使用

**.eslintrc.js**

```js
module.exports = {
  extends: ["@rc5/vue"],
};
```

```jsonc
{
  "scripts": {
    "lint": "eslint . --ext .vue,.ts,.tsx" // or eslint .
  }
}
```

## 规则集

```sh
- "@rc5/core"
- "plugin:vue/vue3-recommended"
- "plugin:prettier/recommended"
